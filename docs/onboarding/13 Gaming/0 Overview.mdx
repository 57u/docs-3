# GamingKit

GamingKit is a set of developer that enable you to easily create web3 games for all
supported platforms, including native, mobile, consoles, browsers, and VR.

- Generate wallets for players in the background, removing the need for disruptive transaction pop-ups entirely.
- Utilize smart contracts and their [extensions](/solidity/extensions) in your favorite game engines such as Unity.
- Create powerful onboarding experiences with [Coinbase Pay](https://help.coinbase.com/en/coinbase/trading-and-funding/coinbase-pay/using-coinbase-pay)
  integrations and [gasless transactions](/glossary/gasless-transactions).
