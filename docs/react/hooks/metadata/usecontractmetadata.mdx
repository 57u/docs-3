Generic hook for reading any data from a smart contract via it&rsquo;s function/view/variable name.

```jsx
import { useContractMetadata } from "@thirdweb-dev/react";
```

## Usage

Provide a contract instance as the argument.

```jsx
import { useContract, useContractMetadata } from "@thirdweb-dev/react";

// Your smart contract address
const contractAddress = "{{contract_address}}";

function App() {
  const { contract } = useContract(contractAddress);
  const { data: metadata } = useContractMetadata(contract);
}
```

## Return Value

The hook's `data` property, once loaded, contains the following properties:

```jsx
name?: string;
description?: string;
symbol?: string;
image?: string;
```
