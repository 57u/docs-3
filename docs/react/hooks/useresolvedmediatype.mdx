Hook for resolving the media type and URL of a given URI (including IPFS URIs).

```jsx
import { useResolvedMediaType } from "@thirdweb-dev/react";
```

:::tip IPFS Media Renderer

The [IPFS Media Renderer](/react/ui-components/mediarenderer) component renders the relevant HTML tag for a given URI, including IPFS URIs.

It supports images, videos, audio files, 3D models, and more out of the box.
:::

## Usage

Provide the URI of the media asset as the argument.

```jsx
import { useResolvedMediaType } from "@thirdweb-dev/react";

function App() {
  const { url, mimeType } = useResolvedMediaType(
    "https://example.com/image.jpg", // URL or URI of the media asset
  );

  return <img src={url} type={mimeType} />;
}
```

## Return Value

The hook returns an object containing two properties: :

- `url`: The fully resolved URL, or `undefined` if the URI is invalid.
- `mimeType`: The mime type of the media, or `undefined` if the URI is invalid.

```jsx
{
  url: string | undefined;
  mimeType: string | undefined;
}
```
