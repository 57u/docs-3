Hook for connecting to a smart contract.

```jsx
import { useContract } from "@thirdweb-dev/react";
```

## Usage

Provide your smart contract address as the first parameter.

Once connected, `contract` will be an instance of your smart contract.

The [ABI](https://docs.soliditylang.org/en/v0.8.17/abi-spec.html) is resolved automatically from our on-chain registry.

```jsx
import { useContract } from "@thirdweb-dev/react";

function App() {
  const { contract, isLoading, error } = useContract("{{contract_address}}");
}
```

:::info Import Smart Contracts
If your smart contract was not deployed using thirdweb, you&rsquo;ll need to import it on the [dashboard](/dashboard).
:::

## Return Value

```jsx
{
  contract?: SmartContract
  error?: Error
  isIdle: boolean
  isLoading: boolean
  isFetching: boolean
  isSuccess: boolean
  isError: boolean
  isFetched: boolean
  isFetchedAfterMount: boolean
  isRefetching: boolean
  refetch: (options: {
      throwOnError: boolean
      cancelRefetch: boolean
  }) => Promise<{SmartContract}>
  status: 'idle' | 'error' | 'loading' | 'success'
}
```
