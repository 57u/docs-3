Hook that requests users to connect their Coinbase Wallet to your dApp.

```jsx
import { useCoinbaseWallet } from "@thirdweb-dev/react";
```

## Usage

```jsx
import { useCoinbaseWallet, Web3Button } from "@thirdweb-dev/react";

function App() {
  const connectCoinbase = useCoinbaseWallet();

  return <button onClick={() => connectCoinbase()}>Connect to Coinbase</button>;
}
```
